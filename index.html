What is 思维导图 ？

思维导图是一种将思维形象化的方法，它运用图文并重的技巧，把各级主题的关系用相互隶属与相关的层级图表现出来，把主题关键词与图像、颜色等建立记忆链接。思维导图充分运用左右脑的机能，利用记忆、阅读、思维的规律，协助人们在科学与艺术、逻辑与想象之间平衡发展，从而开启人类大脑的无限潜能。

② 告诉AI你要什么，如：“生成《离骚》思维导图大纲，用 Markdown 格式”，一定要强调Markdown格式！

③ 点击右上角 “复制” 按钮，保存生成的 Markdown 内容

⑤ 用电脑自带的记事本（Windows）或文本编辑（Mac）新建空白文档

⑥ 将 DeepSeek 生成的内容粘贴进去。将文件后缀从 “.txt” 改为 “.md”。注意，Windows 用户需在文件夹选项中开启 “显示文件扩展名”。

⑦ 进入Xmind，点击 “文件”→“导入”→“Markdown”，选择保存的 “.md” 文件

⑧ 根据自己的喜好，删删改改，美化样式就OK了。

（如果想根据已有内容生成思维导图，在第②步时选择上传文件即可）

Markdown

（1）介绍

Markdown是一种轻量级的标记语言，它使用简单的文本格式来创建富文本内容。由于其简洁性和易读性，Markdown被广泛应用于文档撰写、博客写作、项目说明等领域。在制作思维导图时，Markdown可以快速记录和整理思路，并且可以方便地转换为可视化的思维导图。

（2）在线编辑器（点击进入）

（3）语法

标题表示主题

在 Markdown 中，使用 `#` 来表示标题，`#` 的数量表示标题的级别，从1到6级。在思维导图中，不同级别的标题可以对应不同层级的主题。例如：

# 一级主题
## 二级主题
### 三级主题

列表表示分支

Markdown支持有序列表和无序列表，可用于表示思维导图的分支。

无序列表：使用 `-`、`+` 或 `*` 来表示，例如：

- 分支1
- 分支2
- 分支3

有序列表：使用数字和 `.` 来表示，例如：

1. 分支1
2. 分支2
3. 分支3

嵌套列表表示子分支

通过嵌套列表，可以表示思维导图中的子分支。例如：

- 主分支
  - 子分支1
  - 子分支2
    - 子子分支1

强调突出重点

使用 `*` 或 `_` 来表示斜体，使用 `**` 或 `__` 来表示粗体，可用于突出思维导图中的重点内容。例如：

**重要主题**
*次要主题*

链接和图片丰富内容

链接：使用 `[链接文本](链接地址)` 的格式来创建链接，可用于在思维导图中引用外部资源。例如：

[相关资料](https://example.com)

图片：使用 `![图片描述](图片地址)` 的格式来插入图片，可用于在思维导图中添加可视化元素。例如：

![示例图片](https://picsum.photos/200/300)

表格呈现数据

Markdown 可以使用表格来清晰地呈现数据。表格由表头和内容组成，使用 `|` 分隔列，`---` 分隔表头和内容。例如：

| 表头 1 | 表头 2 |
| ---- | ---- |
| 内容 1 | 内容 2 |

脚注补充说明

脚注可以用于对文本中的内容进行补充说明。在需要添加脚注的地方使用 `[^数字]` 标记，然后在文档的其他位置使用 `[^数字]: 脚注内容` 来定义脚注。例如：

这是一个脚注 [^1]。
[^1]: 脚注的具体内容

任务列表管理任务

任务列表可以用于管理任务，使用 `- [ ]` 表示未完成的任务，`- [x]` 表示已完成的任务。例如：

- [ ] 待办事项 1
- [x] 已完成事项 2
